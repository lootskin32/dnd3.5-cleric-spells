# D&D 3.5 Cleric Spell Manager

A spell management tool built for Dungeons & Dragons 3.5 Edition Cleric players. Track your prepared spells, manage your daily slots, and look cool doing it.

## Features

- **Spell levels 0–10** — Most core rulebook spells included, plus select spells from expansion material.
- **Flexible spell management** — Add and remove spells freely to match your prepared list.
- **New Day with clone support** — Start a fresh day with one click. Clone your previous setup or start from scratch, and navigate back to earlier days whenever you need to.
- **Save / Load** — Persist your spell lists so nothing gets lost between sessions.
- **Excessively animated** — Because looking cool is equally important as functionality. Yes, it's bloated. No, I'm not sorry.

## Disclaimer

This project was entirely vibe-coded. There is no architecture. There are no tests. There is only vibes. It works until it doesn't, and when it doesn't, that's a feature. Use at your own risk — and may your saving throws be high.
